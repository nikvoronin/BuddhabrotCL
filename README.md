# BuddhabrotCL

The Buddhabrot renderer with help of OpenCL and with fast heuristic kernel. Based on Laurent Laborde's (ker2x) project [WinBuddhaOpenCL](https://github.com/ker2x/WinBuddhaOpenCL).
